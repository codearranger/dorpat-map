# Books folder

Drop the three *Seattle Now & Then* PDFs here, named exactly:

- `Now-Then-1-Dorpat.pdf`
- `Now-Then-2-Dorpat.pdf`
- `Now-Then-3-Dorpat.pdf`

When present, clicking any popup on the map will open that book to the
feature's first page in a new tab (using the PDF `#page=N` fragment).

These PDFs are gitignored — they're never pushed to GitHub Pages because
they're copyrighted. The "Open page in the book" link only works when
viewing the map locally, where you have the PDFs on disk.
